# DOTFILES

## Bash Config

1. Clone the repo

```bash
git clone https://github.com/kuwarte/dotfiles.git
```

2. Create Backup (e.g.)

```bash
cp ~/.bashrc ~/.bashrc.backup
```

3. Copy the cloned specific file to replace current config (e.g.)

```bash
cp ~/dotfiles/wsl/.bashrc ~/.bashrc
```
